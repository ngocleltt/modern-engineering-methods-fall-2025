# modern-engineering-methods-fall-2025
Этот репозиторий содержит учебный проект для дисциплины "Современные методы решения инженерных задач". Он демонстрирует навыки моделирования онтологий с использованием OWL, Protégé, Docker, Apache Jena Fuseki и Postman.

# task1 : Базовая онтология вьетнамской кухни
Создана онтология "Вьетнамская кухня" с 4 классами (VietnameseDish, StreetFood, Restaurant, Ingredient), 3 объектными свойствами (hasIngredient, servedAt, hasOrigin) и 6 индивидами. Модель включает блюда (Фо, Бань Ми, Бун Ча), ингредиенты и места происхождения. Проверена на consistency в Protégé, сохранена в Turtle-формате. Пример: Фо связан с рисовая лапша и говядина, происходит из Ханоя.

# task 2 : Семейная онтология с выводом родственных связей
Разработана онтология "Человек" с 7 индивидами (GGParent, GParent, Parent, Child и др.), где каждый связан свойством ИмеетРодителя с уникальным другим индивидом (без самосвязей). OWL-аксиомы включают транзитивные свойства (ИмеетПредка, ИмеетПотомка), инверсные связи и функциональные ограничения. Резонер Protégé автоматически выводит полную родословную: предков через rdfs:subPropertyOf и потомков через owl:inverseOf. DL-запросы подтверждают транзитивное замыкание родительских отношений.

# task3 : Развертывание Jena Fuseki с reasoner-поддержкой и проектирование онтологии
Разработан онтологический модуль рекомендательной системы кулинарных рецептов с логическим правилом SWRL для классификации валидных запросов. Предметная область: подписки пользователей на кухни мира (вьетнамская, итальянская, тайская) и категории блюд (острые, вегетарианские, десерты). Создано 8 классов (Пользователь, Запрос, Блюдо, Кухня и др.), 4 объектных свойства и 16+ индивидов.

Логическое правило SWRL (второй TTL-файл) автоматически классифицирует Запрос как ВалидныйЗапрос если пользователь имеет подписку на кухню/категорию, к которой относится запрашиваемое блюдо. Проверено в резонере Protégé: например, запрос Ле Ньы Нгок на "Фо Бо" признан валидным (подписка на азиатскую кухню → вьетнамская кухня → Фо Бо), а запрос Анны на "Паста Карбонара" — невалидным (нет подписки на итальянскую кухню).
